## Naming of variables, functions, ...

### 1
  ###What
  Name variables, functions etc. as specific as possible.
  ###Why
  This improves the readability of your code for you and other people.

  ###Bad
  ```javascript
  function find(id) {
  }
  ...
  
  ###Good
  ```javascript
  function findCustomer(customerId) {
  }
